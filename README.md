# Name of project
Indentation_Analysis

Description of project
This code analyzes Brinnel like indentation testing on rock samples.

## How to run
1) Create venv: \py -3.12 -m venv .venv\ and activate it.
2) \pip install -r requirements.txt\
3) Put your data files in \data_raw/\
4) Run src\file.py.
